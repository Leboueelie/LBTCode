# LBTCode

[![GitHub License](https://img.shields.io/github/license/Leboueelie/LBTCode?style=flat-square)](https://github.com/Leboueelie/LBTCode/blob/main/LICENSE)
[![Bun](https://img.shields.io/badge/runtime-Bun-black?style=flat-square&logo=bun)](https://bun.sh/)

**LBTCode** — AI Desktop Assistant for THE IT FOUNDATION.

Tauri-based desktop application for chatting with language models (Anthropic, OpenAI, Gemini, Grok), editing code, browsing files, and configuring dev tools.

## Features

- AI Chat with multi-provider support
- Code editor with file open/save
- File explorer with tree navigation
- Settings panel (provider, API key, model)

## Quick Start

```bash
git clone https://github.com/Leboueelie/LBTCode.git
cd lbtcode
bun install
bun run dev
```
