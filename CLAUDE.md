Instructions pour les agents IA travaillant sur LBTCode.

## Projet

LBTCode — Assistant IA Desktop pour THE IT FOUNDATION.
Application Tauri + React + Bun, multi-providers IA.

## Commandes principales

```bash
bun install           # Installer les dépendances
bun run dev           # Backend + Desktop
bun run build         # Build complet
bun run tauri:dev     # Mode Tauri
bun run test          # Tests
```
